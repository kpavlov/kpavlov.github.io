About Me

I have been writing software for more than 15 years. Starting with BASIC on ZX Spectrum and Pascal on x86, moving onto full stack Java / J2EE / GWT and now focusing on Core Java / Spring / Javascript and React.

I believe in agility in software development, practicing S.O.L.I.D. and Test-Driven Development with respect of Ockham’s razor and Pareto principles.

Currently I’m living in Limassol, Cyprus, contracting in financial industry.
